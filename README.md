# 🔍 Python Port Scanner

A fast and simple multithreaded TCP port scanner written in Python.

## ⚙️ Features
- Scans ports 1 to 1024 (can be extended)
- Shows open ports only
- Multithreaded using ThreadPoolExecutor
- Fast and responsive

## 🛠️ Requirements
- Python 3.x

## 🚀 Run the Tool

```bash
python port_scanner.py
